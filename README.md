# Platformer
A 2D  retro-style platformer game created in Unity engine. This game was created as a final project for the AP8PO course.

## Features
* 7 levels
* 4 types of enemies
* Level mechanics like moving platforms, jump pads or smasher
* Game progress saving
* Pause menu
* Level selector
* Collectible items
* Level statistics
